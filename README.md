# Cryptography Lab Assignment

This repository contains implementations of various cryptography algorithms developed as part of cryptography laboratory exercises.

## Implemented Algorithms
- Shift Cipher
- Caesar Cipher
- Vigenère Cipher
- Playfair Cipher
- Rail Fence Cipher
- GCD (Greatest Common Divisor)
- Euler Totient Function
- Multiplicative Inverse
- Coprime Check
- Extended Euclidean Algorithm
- Prime Number Check
- Rabin-Miller Primality Test
- Diffie-Hellman Key Exchange
- RSA Encryption and Decryption
- ElGamal Cryptosystem
- Malicious Logic Demonstration
- DES Algorithm

## Purpose
To understand and implement fundamental cryptographic techniques and number theoretic algorithms used in modern security systems.
